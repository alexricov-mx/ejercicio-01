Nuestra misión

Este sitio nació como parte del curso de Python con Django. La idea es practicar vistas, templates, URLs y poco a poco ir sumando funcionalidades: posts, formularios, usuarios y más.

Si estás empezando, lo importante no es tener un diseño perfecto, sino entender cómo Django conecta cada pieza: url → vista → template → respuesta HTML.
